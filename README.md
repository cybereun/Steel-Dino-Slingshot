> **기계 군단 로봇 요새를 격파하라! 강철 공룡들의 강력한 투석 물리 액션 디펜스 퍼즐 게임**
>
> 본 프로젝트는 플레이어가 강철 공룡들을 투석기(Slingshot)로 쏘아, 정교한 물리 구조물 벙커 속에 숨어있는 기계 로봇 군단을 격파하는 2D 물리 시뮬레이션 디펜스 게임입니다.

---

## 🎮 게임 소개
**Steel Dino Slingshot**은 래비님을 위해 특별히 맞춤 제작된 독창적인 퍼즐 물리 액션 게임입니다. 다양한 성질의 블록(나무, 돌, 유리, 폭발성 TNT)으로 이루어진 요새 구조물을 파괴하고, 요새를 장악하고 있는 적 로봇들을 모두 섬멸해야 합니다. 각 공룡이 가진 고유한 액티브 스킬과 4가지 아름다운 스킨 테마를 경험해 보세요!

---

## ✨ 주요 특징 (Features)

*   **Matter.js 기반 실시간 2D 물리 엔진**:
    *   중력, 질량, 마찰력, 탄성 계수가 세밀하게 적용되어 충돌 각도와 세기에 따라 사실적으로 벽이 무너지고 연쇄 붕괴가 일어납니다.
*   **3종 특화 강철 공룡 히어로 (Dino Heroes)**:
    *   🦖 **티라노 (Tyranno)**: 무거운 질량으로 돌파하며, 공중에서 터치 시 **대형 폭발**을 일으켜 주변 블록과 적을 한꺼번에 소멸시킵니다.
    *   🐉 **트리케라 (Tricera)**: 돌진 특화 공룡. 공중에서 터치 시 **초고속 황금 돌진**을 가해 단단한 돌 벽도 단숨에 뚫고 지나갑니다.
    *   🦅 **프테라 (Ptera)**: 공중에서 터치 시 **3개의 개별 포탄으로 분열**되어, 여러 곳의 타겟을 광범위하게 동시 타격합니다.
*   **30단계의 정교한 스테이지**:
    *   `🌿 쉬움 (1~10 STAGE)`, `⚡ 보통 (11~20 STAGE)`, `💀 어려움 (21~30 STAGE)`으로 나뉘는 총 30개의 다채로운 스테이지가 준비되어 있습니다.
    *   각 스테이지마다 최고 점수 및 2성/3성 달성 커트라인이 기록되어 한계 돌파가 가능합니다.
*   **실시간 배경 테마(스킨) 선택 시스템**:
    *   🌌 **우주 야경 (Space Night)**: 반짝이는 은하계 별빛 입자 물리 효과가 가미된 은은한 밤하늘 테마.
    *   ☀️ **에메랄드 낮 (Emerald Day)**: 뭉게구름이 흘러가고 에메랄드빛 잔디 구릉이 시원하게 보이는 매우 밝고 청명한 테마.
    *   🌅 **노을 황혼 (Sunset Glow)**: 붉게 타오르는 실루엣 산맥 뒤로 웅장한 태양이 반투명하게 비치는 따뜻하고 평온한 테마.
    *   🔮 **사이버 네온 (Cyberpunk Neon)**: 핫 핑크와 마젠타 빛 레트로 원근 그리드가 캔버스 하단에 깔리고 네온 달이 빛나는 미래지향적 테마.
    *   *밝은 '낮(day)' 테마 활성화 시, 조준선 포물선 및 텍스트 게이지 색상이 대비감 있는 어두운 딥 블루/레드로 자동 변환되어 시인성을 완벽히 보장합니다.*
*   **입체적인 오디오 연출**:
    *   투석기 고무줄을 당길 때 늘어나는 인장음, 발사음, 공중 비행 시 속도와 비례하여 변조되는 바람 소리 루프, 돌/나무/유리 재질별 충돌 파쇄음 및 TNT 폭발 사운드가 탑재되어 묵직한 타격감을 자랑합니다.

## 📜 개발 역사 (History)
이 게임은 래비님의 적극적인 피드백을 통해 계속해서 업데이트되며 완성되었습니다.

1.  **자동 발사 오작동 해결**: 게임 시작과 동시에 강제로 발사되어 곧바로 게임이 끝나던 상태를 마우스/터치 드래그 대기 턴제로 개편.
2.  **예상 궤적 가이드 및 조준 파워**: 조준 방향에 따라 닷 구슬 궤적 포물선 및 LAUNCH POWER(%) 텍스트를 연동하여 전략적인 정밀 조종 기반 확립.
3.  **에셋 크로마키 투명화**: 공룡 캐릭터와 프롭 에셋의 투명화 가장자리 마젠타(#ff00ff) 배경색을 로딩 시 자동으로 지워 투명화 테두리로 보정.
4.  **입체적 사운드 연동**: Web Audio API로 구현한 다이내믹 사운드 시스템을 적용하여 게임의 흥미 요소를 대폭 증가.
5.  **30단계 확장 & 로비 난이도 탭**: 게임 분량을 두 배인 30단계로 늘리고, 로비에 난이도별(`전체`, `쉬움`, `보통`, `어려움`) 필터 탭 UI를 추가하여 세련된 레이아웃 제공.
6.  **다이내믹 배경 스킨 기능**: 배경이 어둡다는 피드백을 반영하여 캔버스 자체 렌더링 방식의 4종 실시간 변경 스킨 및 눈 피로도 감소를 위한 가독성 자동 보정 적용.

## 🛠️ 제작 기반 (Tech Stack)

*   **Core**: React 18, TypeScript, Vite
*   **Physics Engine**: Matter.js
*   **Rendering**: HTML5 Canvas (라이브 애니메이션 렌더링)
*   **Audio Engine**: Web Audio API (합성 이펙트 오디오 시스템)
*   **Styling**: Vanilla CSS (Premium Glassmorphism & Neon Glow)

## 🎮 사용법 (How to Play)

1.  로비에서 도전하고자 하는 스테이지 카드의 `전투 개시 ⚔️` 버튼을 클릭합니다.
2.  새총 위에 대기 중인 공룡 영웅을 마우스 클릭 또는 손가락 터치로 당겨 조준합니다. 당긴 각도의 반대 방향으로 포탄이 날아갑니다.
3.  화면에 표시되는 포물선 조준 궤적과 상단 `LAUNCH POWER: %`를 조절하며 손을 놓아 발사합니다.
4.  공룡이 날아가는 동안 **화면 아무 곳이나 마우스 클릭 또는 터치**를 가하면 해당 공룡 고유의 **액티브 스킬**이 가동됩니다.
    *   *예: 돌 벽 뒤에 있는 적에게는 트리케라로 벽을 뚫고 지나가거나, 밀집된 적 근처에서 티라노로 폭발 스킬을 사용해 보세요!*
5.  공룡이 소진되기 전 구조물을 부숴 요새 안의 모든 적 로봇(🤖)을 격파하면 작전 성공(STAGE CLEAR)입니다.
6.  게임 중이라도 상단 바의 미니 스킨 버튼(🌌, ☀️, 🌅, 🔮)을 눌러 실시간으로 배경을 변경하여 즐길 수 있습니다.
